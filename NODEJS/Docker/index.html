Introducción a Docker

¿Qué es Docker?

Docker es una plataforma de contenerización que permite empaquetar, distribuir y ejecutar aplicaciones en entornos aislados llamados contenedores...

Conceptos Clave

  • Imagen: Un paquete ejecutable que incluye todo lo necesario para ejecutar una aplicación, incluidas bibliotecas, dependencias y código...
  • Contenedor: Una instancia en ejecución de una imagen. Los contenedores son entornos aislados que ejecutan aplicaciones y comparten el núcleo del sistema operativo del host...
  • Dockerfile: Un archivo de configuración que especifica cómo se debe construir una imagen. Contiene instrucciones para instalar software, configurar el entorno y definir la aplicación...

Instalación de Docker

Linux

        
sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io
sudo usermod -aG docker $USER

macOS

Descarga e instala Docker Desktop.

Windows

Descarga e instala Docker Desktop.

Comandos Básicos de Docker

  1. Verificar instalación:

    docker --version
    docker-compose --version

  2. Ejecutar un contenedor:

    docker run hello-world
