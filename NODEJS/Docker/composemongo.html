Guía Docker Compose con MongoDB y Mongo Express

Esta guía te mostrará cómo crear una conexión con MongoDB y un administrador de base de datos (Mongo Express) utilizando Docker Compose.

Paso 1: Crear un archivo docker-compose.yml


version: '3'
services:
  mongo:
    image: mongo:latest
    ports:
      - "27017:27017"
    environment:
      MONGO_INITDB_ROOT_USERNAME: root
      MONGO_INITDB_ROOT_PASSWORD: example

  mongo-express:
    image: mongo-express:latest
    ports:
      - "8081:8081"
    environment:
      ME_CONFIG_MONGODB_SERVER: mongo
      ME_CONFIG_MONGODB_ADMINUSERNAME: root
      ME_CONFIG_MONGODB_ADMINPASSWORD: example

Este archivo define dos servicios: mongo para el servidor MongoDB y mongo-express para el administrador de MongoDB (Mongo Express). Ajusta las variables de entorno según tus preferencias.

Paso 2: Ejecutar Docker Compose

Guarda el archivo como docker-compose.yml y ejecuta el siguiente comando en la misma carpeta que el archivo: 


docker-compose up

Este comando descargará las imágenes necesarias y creará los contenedores de MongoDB y Mongo Express según la configuración del archivo docker-compose.yml.

Iniciar creacion

docker-compose -f mongo.yml up

Paso 3: Acceder a Mongo Express

Abre tu navegador web y visita http://localhost:8081. Verás la interfaz de inicio de sesión de Mongo Express.

  • Usuario: root
  • Contraseña: example
  • Servidor: mongo

Ahora estarás conectado a Mongo Express y podrás administrar la base de datos MongoDB desde una interfaz web.

¡Listo! Has creado una conexión con MongoDB y un administrador de base de datos utilizando Docker Compose. Este entorno es ideal para el desarrollo y prueba de aplicaciones con MongoDB. Recuerda ajustar las configuraciones según tus necesidades específicas.
