Dockerizando MySQL y phpMyAdmin

En esta guía, aprenderás cómo crear una aplicación dockerizada con MySQL y phpMyAdmin. Esto te permitirá tener un entorno de desarrollo de base de datos MySQL de forma rápida y portable.

Pasos

  1. Paso 1: Ejecutar un Contenedor MySQL

    Abre tu terminal y ejecuta el siguiente comando para descargar y ejecutar un contenedor de MySQL: 

    docker run --name mi-mysql-container -e MYSQL_ROOT_PASSWORD=my-secret-pw -d mysql:latest
    • --name mi-mysql-container: Establece el nombre del contenedor como "mi-mysql-container".
    • -e MYSQL_ROOT_PASSWORD=my-secret-pw: Especifica la contraseña del usuario root de MySQL.
    • -d: Ejecuta el contenedor en segundo plano.

  2. Paso 2: Ejecutar un Contenedor phpMyAdmin

    Ejecuta el siguiente comando para descargar y ejecutar un contenedor de phpMyAdmin: 

    docker run --name mi-phpmyadmin -d --link mi-mysql-container:db -p 8080:80 phpmyadmin/phpmyadmin
    • --name mi-phpmyadmin: Establece el nombre del contenedor como "mi-phpmyadmin".
    • --link mi-mysql-container:db: Establece una conexión entre el contenedor de phpMyAdmin y el contenedor de MySQL.
    • -p 8080:80: Mapea el puerto 8080 de tu máquina local al puerto 80 del contenedor de phpMyAdmin.

  3. Paso 3: Conectar phpMyAdmin a MySQL

    Abre tu navegador web y visita http://localhost:8080. Ingresa las credenciales:

    • Usuario: root (o según la configuración de tu MySQL).
    • Contraseña: my-secret-pw (o la contraseña que hayas establecido).
    • Servidor: db (nombre del host que estableciste al enlazar los contenedores).

    Ahora estarás conectado a phpMyAdmin y podrás administrar la base de datos MySQL desde una interfaz web.
